# reisebuero-website
Globetrotter Reisebüro ist eine moderne, interaktive Website für ein fiktives Reisebüro, das Kunden eine Übersicht über Reiseziele, Angebote und Dienstleistungen bietet. Die Website ist für Desktop und mobile Geräte optimiert und bietet ein modernes, ansprechendes Design mit interaktiven Elementen.
